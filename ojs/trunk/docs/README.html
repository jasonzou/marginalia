Marginalia for OJS Readme

This release of Marginalia is for Open Journal Systems. It allows users to add margin notes to journal articles. For updates to this release and for a stand-alone demo version of the code, please see my web site. Developers should see the Subversion repository for test scripts, utilities, etc.

Please read INSTALL.html to get started, and CREDITS.html for information about the organizations and individuals who have funded, supported, and helped with this project.
